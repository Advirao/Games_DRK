# Harry Potter Character Guesser

A simple browser game where you guess Harry Potter characters from clues.

## How to play

1. Open `index.html` in your browser.
2. Click **Start Game**.
3. Read the clue and pick the correct character from the four options.
4. You have 3 lives. Wrong answers cost a life; correct answers add to your score.
5. Game ends when you run out of lives. Click **Play Again** to start over.

## Files

- `index.html` – Main page
- `game.js` – Game logic and character data
- `style.css` – Hogwarts-themed styling

No build step or server required—just open the HTML file.
